### 자료 구성 안내

- **pre-report**: 논문을 읽고 핵심 내용을 정리한 사전 리뷰
- **implementation**: 논문의 핵심 아이디어를 검증하기 위해 직접 구현한 코드 (Jupyter Notebook)
- **final-report**: 구현 및 실험 결과를 바탕으로 작성한 최종 결론 보고서

### 참고

- 논문 원문은 저작권 문제로 포함하지 않으며, 각 보고서 내 참고문헌에 원문 링크를 명시합니다.
- 구현 코드는 논문의 핵심 아이디어를 검증하기 위한 축소/재현 버전이며, 원 논문의 전체 스케일과 다를 수 있습니다.

### File Types

- **pre-report**: Initial summary written after reading the paper
- **implementation**: Hands-on code (Jupyter Notebook) built to verify the paper's core ideas
- **final-report**: Final conclusions based on implementation and experiment results

### Notes

- Original paper texts are not included due to copyright; each report cites the original paper link in its references.
- Implementations are scaled-down/reproduction versions intended to verify core ideas, and may differ from the original paper's full scale.
